 Kisan – Farmer to Consumer Marketplace

Kisan is a full-stack web application that directly connects **farmers** and **consumers**, eliminating middlemen and ensuring **fair pricing, transparency, and trust**.  
The platform enables farmers to list products, consumers to place orders, negotiate prices, and track order status in real time.

---

##Features

## Farmer
- Add and manage agricultural products
- View incoming consumer orders
- Approve / reject orders
- Negotiate prices with consumers
- Track order and payment status

##Consumer
- Browse farmer-listed products
- Place orders with requested quantity
- Negotiate price with farmers
- Track order lifecycle (requested → approved → paid → completed)

##Authentication
- Secure login & signup
- Role-based access (Farmer / Consumer)

## Additional Highlights
- Real-time order visibility
- Clean and responsive UI
- Scalable backend architecture
- Ready for AI-based recommendation integration (future scope)
